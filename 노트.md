# 시스템 UI 메모

## 테이블 대기 구역(=궤도 구역)

- **정의**: 사용한 카드가 즉시 사라지지 않고, 필드에 머무는 **시각적 대기 공간**

### 배치 & 상태 변화

- **사용 직후 (Active)**: 중앙 필드에서 효과 발동
- **1턴 경과 (Cooling)**: 플레이어 앞 **‘궤도 구역’ 슬롯**으로 이동 → **반투명** 표시
- **2턴 종료 (Returning)**: 카드가 **테이블 덱 내부 랜덤 위치**로 흡수

### 시각적 가이드

- **슬롯 시스템**: 핸드와 덱 사이 공간에 **소형 카드 슬롯** 배치ㅌ
- **카운트다운**: 카드 모서리에 **덱 복귀까지 남은 턴 수** 표시

### 예외 로직

- **소멸 카드**: 궤도 구역으로 이동하지 않고 **산산조각** 연출로 소멸
    - 단, **심판 카드**에 의해 소멸 카드가 부활한 경우: 대기 구역 이동 없이 **즉시 카드 능력대로 사용**

---

## 역방향 디버프(손패 리스크)

- **정의**: 역방향 카드를 쥐고 있을 때 플레이어가 감수해야 하는 누적 패널티

### 단계

1. **유예 단계 (Grace)**: 카드를 처음 획득한 턴에는 패널티 없음
2. **활성 단계 (Cursed)**: 다음 턴부터 테두리에 **어두운 오라** → 패널티 누적

### 디버프 명세

| **역방향 카드 수** | **디버프 명칭** | **효과 상세** |
| --- | --- | --- |
| 1장 | 운명의 침식 | 턴 종료 시 **HP -5** |
| 2장 | 영혼의 부식 | 턴 종료 시 **HP -10** (누적 총 -15) |
| 3장 | 자원의 고갈 | 턴 종료 시 HP 피해 + **다음 턴 시작 코스트 -1** |
| 4장 | 인과의 차단 | HP 피해 및 코스트 감소 + **다음 턴 드로우 불가** |
| 5장 이상 | 파멸의 예언 | 매 턴 종료 시 **HP 10% 감소** 및 모든 행동 제약 |
- **해제 방법**: 역방향 카드를 직접 사용하거나, 특정 카드로 **소모/정화**

## 카드 특수 상태

### 1) 마법사 계약 카드

- **정의**: 마법사의 효과로 지정되어 **1턴 동안 위력 2배 + 코스트 0**
- **시각적 연출**: 테두리 광원 효과
- **UI 요소**: 위력 옆 **x2** 아이콘, 코스트 숫자 **황금색 고정**
- **상태 표시**: 마법사 카드가 소멸해도 계약 상태를 보여주는 **워터마크**

### 2) 환영 카드

- **정의**: 마법사(역방향)로 복사된 카드, **발동 확률 50%**
- **시각적 연출**: **반투명(Opacity 60~70%)**, 보라색 연기 효과
- **UI 요소**: 카드 중앙 또는 코스트 자리에 큰 **'?'** 마크
- **실패 연출**: 뒤집힘/타버림 대신 **안개처럼 흩어짐**

### 3) 복제 카드

- **정의**: 광대 카드로 복제됨 → **코스트 0**, **효과 50%**
- **시각적 연출**: 흑백 처리 또는 구석에 **'Copy' 스탬프**
- **UI 요소**: 기존 수치에 **취소선**, 절반 값으로 수정 표기

### 4) 능력 사용 카드

- **정의**: 사용되어 효과가 발동 중인 카드
- **시각적 연출**: 회색조(비활성 느낌)
- **강조**: 지속 효과 적용 중이면 카드 주변에 **은은한 파동**

## 툴팁 시스템

### 1) 기본 구조 (Basic Structure)

- **카드 이름 및 수트**: (Major / Wands / Swords 등) 최상단
- **기본 효과 설명**: 정방향/역방향 효과를 구분해 텍스트로
- **키워드 하이라이트**: ‘소멸’, ‘드류오’, ‘코스트’ 등 특수 용어를 색상 강조 + 필요 시 **중첩 툴팁**

### 2) 상태별 동적 정보 (Dynamic Status Info)

- 카드가 특수 상태일 때, 툴팁 하단에 **‘상태 정보’ 섹션** 추가

| **상태** | **툴팁 추가 문구** | **비고** |
| --- | --- | --- |
| 마법사의 계약 | "✨ **계약됨**: 이번 전투 동안 위력 2배 및 코스트 0" |  |
| 환영 (마법사 역) | "🌫️ **환영**: 발동 시 50% 확률로 효과가 무효화됨" |  |
| 복제 (광대) | "🃏 **복제본**: 코스트 0, 모든 효과가 50%로 감소함" |  |
| 역방향 (유예) | "⏳ **운명의 유예**: 이번 턴에는 페널티가 발생하지 않음" |  |
| 역방향 (저주) | "💀 **저주 활성**: 턴 종료 시 보유 장수만큼 HP 감소 페널티 발생" |  |
| 궤도 대기 (테이블) | "🔄 **운명의 궤도**: n턴 뒤 드로우 더미로 랜덤하게 섞여 들어감" |  |

### 3) 상호작용 & 노출 로직

- **지연 시간 (Delay)**: 마우스 오버 후 **0.2~0.3초** 뒤 노출
- **스마트 포지셔닝**: 하단(핸드)=위쪽 / 중앙=측면 등 동적 조정
- **전투 로그 연동**: 툴팁에서 클릭(또는 특정 키) 시 최근 사용 기록/관련 룰을 상세 창으로 확장 가능

# 덱/카드 구성 메모

## 아르카나 구성

- 마이너 아르카나: **56장**
- 메이저 아르카나: **공유 카드**

### 마이너 아르카나 구조

- **14 x 4**
- 수트: **wands / swords / cups / coins**
- 랭크: **A, 2, 3, 4, 5, 6, 7, 8, 9, P, N, Q, K**

### 수트 테마(트럼프/포커 대응)

- 막대기/지팡이 → 모험, 야망, 정열, 용기 → **클로버**
- 검 → 패배, 곤란, 손실, 이별, 커뮤니케이션 → **스페이드**
- 잔/성잔 → 애정, 감정, 꿈, 가족 → **하트**
- 동전/디스크 → 경험, 실적, 재산, 지위 → **다이아몬드**

---

## 덱 선택/시작 흐름(메모)

- 게임 시작 전에 **마이너 아르카나 덱을 하나 선택**
- 여기서 14장씩 4개를 만들면 → **하나의 덱**
- 선택한 덱 + 공유 아르카나 36장

### 역방향 카드(요약)

- 처음 카드를 얻은 턴: **1턴 유예**
- 다음 턴부터: 보유한 **역방향 카드 개수만큼 디버프 중첩**

### 시작 핸드(메모)

- 시작 → 5장 얻음 → **메이저 2장 + 마이너 3장**
- 마이너: 랜덤 / 메이저: ?
- 메이저를 얻음 → 역방향 감지 → ?

## 2. 카드 구조 (Anatomy)

```
┌──────────────────────┐
│  ☀  ✦  ⊕  ✦  ✦  ✦  │  ← 상단 천체 장식 (태양, 별, 행성)
│ ╔══════════════════╗ │
│ ║                  ║ │ ← 금색 이중 테두리
│ ║   ♈             ║ │ ← 좌측: 점성술 기호 수직 배열
│ ║   ♄    ARTWORK  ║ │
│ ║   ♃             ║ │   (전체 카드 높이의 약 78%)
│ ║                  ║ │
│ ║                  ║ │ ← 우측: 수트별 심볼 (검, 지팡이 등)
│ ╠══════════════════╣ │
│ ║   VII            ║ │ ← 로마 숫자 (카드 번호)
│ ║ ─ THE CHARIOT ─ ║ │ ← 카드 이름 (금색 세리프, 좌우 장식선)
│ ╚══════════════════╝ │
│        ☽             │  ← 하단 초승달 장식
└──────────────────────┘
```

### 영역 비율

| 영역 | 카드 높이 대비 비율 |
| --- | --- |
| 상단 장식 (천체) | 상단 4% |
| 아트워크 | 상단 4% ~ 78% |
| 텍스트 영역 (번호 + 이름) | 하단 18% |
| 하단 장식 (초승달) | 하단 4% |

### 테두리 & 장식 규칙

- **이중 테두리**: 외곽 굵은 금선 + 내측 세선, 사이 간격 3~4px
- **모서리 장식**: 금색 꽃/잎 문양 필리그리
- **좌측 심볼**: 해당 카드에 연관된 점성술 기호 2~3개를 수직으로
- **텍스트 폰트**: 로마 숫자는 크고 굵게 / 카드 이름은 자간 넓은 소형 세리프 대문자
- **배경색**: 순수 검은색 (`#000000`) 기반, 아트워크와 자연스럽게 블렌딩

## 3. 핸드 레이아웃 (부채꼴 배치)

### 기본 규칙

- 화면 **하단 중앙** 기준점으로 부채꼴 전개
- 카드 수량: 최소 1장 ~ 최대 10장
- 카드 간 겹침: 카드 너비의 **40~50%** 겹침
- 전체 펼침 각도: **-18° ~ +18°** (총 36° 범위), 카드 수에 따라 동적 조정

### 각도 & 위치 계산

| 카드 수 | 카드당 각도 간격 | 최대 기울기 |
| --- | --- | --- |
| 1~3장 | 6° | ±6° |
| 4~6장 | 5° | ±12° |
| 7~10장 | 4° | ±18° |

- 중앙 카드는 **수직(0°)**, 좌우로 갈수록 기울기 증가
- 각 카드는 부채꼴 호를 따라 위치 — 중앙에서 멀수록 살짝 아래로

### 인터랙션

| 상태 | 처리 |
| --- | --- |
| 기본 | 부채꼴 정렬, 겹침 유지 |
| 마우스 오버 | 해당 카드 위로 24px 상승 + 앞으로(Z) 올라옴 + 0.15초 이징 |
| 선택 클릭 | 화면 중앙 필드로 이동 애니메이션 |
| 드래그 | 카드가 손에서 분리되어 드래그, 필드 위 놓으면 사용 |

## 4. 역방향 카드 표시

- 카드 이미지 **180° 회전** 표시
- 테두리 오라 효과: 디버프 단계별로 색상/강도 변화

| 단계 | 상태 | 오라 표현 |
| --- | --- | --- |
| 유예 | Grace | 오라 없음, 테두리만 옅은 보라색 |
| 1장 | Cursed Lv.1 | 어두운 보라색 오라, 낮은 강도 |
| 2장 | Cursed Lv.2 | 보라색 오라 + 미세 맥동 |
| 3장 | Cursed Lv.3 | 어두운 붉은 오라 + 맥동 강화 |
| 4장 | Cursed Lv.4 | 검붉은 오라 + 틱마다 흔들림 |
| 5장+ | Doom | 검은 불꽃 오라, 화면 가장자리까지 영향 |

---

## 5. 카드 상태별 시각 표현 (요약 통합)

| 상태 | 시각 처리 |
| --- | --- |
| **마법사 계약** | 테두리 황금 광원 + 위력 옆 ×2 아이콘 + 코스트 황금색 고정 |
| **환영** | Opacity 60~70% + 보라색 연기 파티클 + 중앙 '?' 마크 |
| **복제 (광대)** | 흑백 처리 또는 우하단 'COPY' 스탬프 + 수치 취소선 후 50% 값 |
| **능력 사용 중** | 회색조 처리 + 지속 효과 시 카드 주변 은은한 파동 |
| **궤도 대기 (Cooling)** | Opacity 50% + 소형(80×167) + 모서리 카운트다운 숫자 |
| **소멸** | 산산조각 파티클 연출 후 제거 (궤도 구역 없음) |
| **역방향 유예** | 180° 회전 + 옅은 보라 테두리 |
| **역방향 저주** | 180° 회전 + 단계별 오라 (위 4번 참조) |

---

## 6. 비주얼 컨셉 요약

- **배경**: 순수 검정 (`#000000`)
- **강조색**: 금색 (`#C9A84C` 계열)
- **보조색**: 짙은 보라 (역방향/저주), 은색 (비활성/복제)
- **키워드**: 점성술, 타로, 블랙 & 골드
- **폰트**: 세리프 + 넓은 자간 (영문), 명조/바탕 계열 (한글)
- **파티클**: 별빛 반짝임, 연기, 오라 맥동 — 과하지 않게 절제
